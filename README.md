# ☀️ Irradiância Solar - por Horário

Aplicação interativa desenvolvida em **Python + Streamlit** que permite visualizar a **irradiância solar** por intervalo de horário em diferentes regiões.  
O projeto gera **gráficos de bolhas coloridas** e **animações em GIF**, mostrando os valores máximos de irradiância (W/m²) em 6 faixas horárias.
